# Smart Kitchen Logistics（SKL）
## 〜調理師×ITによる厨房DXプロトタイプ〜
---
## 📋 プロジェクト概要
老人ホーム厨房の現場経験（原価率74.5%維持）をJavaでシステム化したプロトタイプです。
PDF・Excel・DBからのデータ読み込みを基盤に、AI連携による厨房DXの実現を目指しています。
---
## 🎯 解決する課題
| 現場の問題 | システムによる解決 |
|-----------|-----------------|
| 喫食数・請求の手作業管理 | 自動計算・請求書生成 |
| 原価率の把握が遅れる | リアルタイム原価率表示 |
| 外国人スタッフへの指示が困難 | 多言語音声ナビゲーション |
| 残業・メンタルケアが属人的 | AIによる自動管理・ケア |
---
## 💰 業務データ
- 朝食単価：460円 / 昼食単価：590円 / 夕食単価：590円
- 管理費売上：2,040,000円 / 月
- 目標食材原価率：70%
- 残業アラート：20時間超過で自動通知
